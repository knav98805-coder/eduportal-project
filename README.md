# 🎓 EduPortal Project

A modern, responsive educational portal built with **Vite**, **React**, and **TypeScript**. Designed for speed, modularity, and seamless AI integration.

---

## 🚀 Features

- ⚡️ Vite-powered dev environment
- 🧠 AI backend integration (Ollama / Gemini)
- 🎨 TailwindCSS for styling
- 🧩 Modular component architecture
- 🌐 Deployed on Vercel

---


## 📦 Tech Stack

### Frontend
- React
- TypeScript
- TailwindCSS

### Backend (Optional)
- Express.js
- Ollama

### APIs
- Gemini API (fallback)

### Tooling & DevOps
- Vite
- ESLint
- Prettier
- GitHub
- Vercel (Deployment)
